# 🎓 CampusLoop - Smart Closed-Campus Marketplace

CampusLoop is a premium, modern marketplace platform designed specifically for university ecosystems. It facilitates secure buying, selling, and networking within the student community.

🚀 **Live Demo:** [https://campusloop.onrender.com](https://campusloop.onrender.com)

---

## ✨ Features

- **🔐 Secure Authentication**: Integrated student-only access with role-based security.
- **🛒 Dynamic Marketplace**: Browse, search, and filter products by category, price, and condition.
- **💬 Real-time Interaction**: Built-in chat system for negotiation and inquiries.
- **📍 Hyper-local Logistics**: Micro-location selection (Hostels/Departments) for easy exchanges.
- **⭐ Reputation System**: User ratings and reviews to build trust within the campus.
- **🤖 Smart Assistant**: Integrated chatbot to help students navigate the marketplace.

---

## 🛠️ Technology Stack

- **Backend**: Spring Boot 3.2.5 (Java 17)
- **Security**: Spring Security 6
- **Database**: H2 (Demo Mode) / MySQL Support
- **Frontend**: Thymeleaf, Modern Vanilla CSS (Glassmorphism design)
- **Deployment**: Docker, Render, GitHub Actions

---

## 🚀 Getting Started (Local Development)

### Prerequisites
- JDK 17 or higher
- Maven 3.8+

### Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/Rahulmani421/campus-loop.git
   cd campus-loop
   ```
2. Build the project:
   ```bash
   mvn clean install
   ```
3. Run the application:
   ```bash
   mvn spring-boot:run
   ```
4. Access the app at: `http://localhost:8080`
